# Church Office
This is the frontend application used by the Berlin.Church.


# Installation

```sh
npm install
```

# Running

After installing you can simply run it via:

```sh
npm run dev
```

# Development

This application uses ReactJS written in Typescript and the authentication is done via Auth0.
